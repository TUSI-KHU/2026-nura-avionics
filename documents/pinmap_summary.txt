    NURA current stack PCB pinmap summary
Generated from the verified Teensy 4.1 smoke-test pinmap.

Scope
- Carrier PCB for module breakouts.
- All sensor/module VCC pins use Teensy 3.3V, not VIN.
- All GND pins share the same common ground.
- I2C bus is Wire1: SDA1=D17, SCL1=D16, 100 kHz.
- SPI bus is shared: MOSI=D11, MISO=D12, SCK=D13.

Global power
- +3V3: Teensy 3.3V rail to all sensors/modules.
- GND: common ground to all sensors/modules.
- Do not feed 5V/VIN into LIS3MDL, MPL3115A2, LSM6DSO32, H3LIS331DL, or SX1278 VCC.

I2C bus, confirmed working
- Teensy D17 / SDA1 -> I2C_SDA
- Teensy D16 / SCL1 -> I2C_SCL
- R1: 4.7 kOhm from I2C_SDA to +3V3
- R2: 4.7 kOhm from I2C_SCL to +3V3
- Only one pull-up pair is required on the bus.

LIS3MDL magnetometer, I2C address 0x1C
- VIN/VCC -> +3V3
- GND -> GND
- SCL -> Teensy D16 / SCL1
- SDA -> Teensy D17 / SDA1
- CS -> +3V3. This forces I2C mode.
- DO/SDO -> GND. This selects I2C address 0x1C.
- Top/additional GND pad -> GND.
- Critical PCB rule: CS must not float; DO/SDO must not float.

MPL3115A2 barometer, I2C address 0x60
- VIN/VCC -> +3V3
- GND -> GND
- SCL -> Teensy D16 / SCL1
- SDA -> Teensy D17 / SDA1

LSM6DSO32 low-g IMU, SPI
- VIN/VCC -> +3V3, not VIN/5V
- GND -> GND
- SCL/SCK -> Teensy D13 / SPI_SCK
- SDA/SDI/MOSI -> Teensy D11 / SPI_MOSI
- DO/SDO/MISO -> Teensy D12 / SPI_MISO
- CS -> Teensy D6
- INT1 -> Teensy D5, optional/reserved
- INT2 -> Teensy D4, optional/reserved

H3LIS331DL high-g accelerometer, SPI
- VIN/VCC -> +3V3
- GND -> GND
- SCL/SCK -> Teensy D13 / SPI_SCK
- SDA/SDI/MOSI -> Teensy D11 / SPI_MOSI
- SA0/SDO/MISO -> Teensy D12 / SPI_MISO
- CS -> Teensy D7
- INT1 -> Teensy D3, optional/reserved
- INT2 -> Teensy D8, optional/reserved

SX1278 / Ra-01 LoRa, SPI
- VCC -> +3V3
- GND -> GND
- SCK -> Teensy D13 / SPI_SCK
- MOSI -> Teensy D11 / SPI_MOSI
- MISO -> Teensy D12 / SPI_MISO
- NSS/CS -> Teensy D10
- RST -> Teensy D9
- DIO0 -> Teensy D2
- DIO1/DIO2 are not used by the current smoke test.

GY-GPS6MV2 / u-blox M6 GPS, UART Serial3
- VCC -> +3V3
- GND -> GND
- GPS TX -> Teensy D15 / RX3
- GPS RX -> Teensy D14 / TX3

Smoke-test result represented by this pinmap
- I2C: LIS3MDL at 0x1C PASS, MPL3115A2 at 0x60 PASS
- SPI: LSM6DSO32 PASS, H3LIS331DL PASS, SX1278 PASS
- UART: GPS NMEA PASS

PCB notes before fabrication
- The KiCad board is a module-carrier wiring template using generic 2.54 mm headers.
- Verify each breakout module's physical pin order before ordering a PCB.
- Keep the SX1278/Ra-01 3.3V rail short and decoupled; LoRa TX current pulses are large.
- Put at least 100 nF decoupling near each module VCC/GND pair.
- Keep I2C traces short and route SDA/SCL together away from LoRa RF output.
